# View system logs

The System Logs section of the Dashboard tab displays troubleshooting‑related logging activity such as Errors, Warnings or Bugs. When you hover your cursor over a log entry, you’ll see additional details about that entry.

## How to use the System Logs section

1. Go to **Admin Panel** > **Dashboard** > **System Logs**.
2. To delete log entries, check the box next to the desired logs and press the “**Delete Selected Entries**” button at the top right.
3. To change the default log level (Error, Warn, Debug, or None), go to **Admin Panel** > **Settings** > **System Tab**.
